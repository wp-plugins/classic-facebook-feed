Easily Display custom feeds Facebook Feeds to WordPress blog site from any Facebook Fan Page.

== Description ==
Classic Facebook Feed is a plugin for display facebook feed on your blog or site. Here you can Get feed fron any Facebook Page. This plugin use Facebook Opengraph API so its highly customizable and you can style the feeds the way you like.

<h4>Features :</h4>	

1. Easily add Facebook Feed using the shortcode
2. Display Facebook Page Feed
3. Set Limit Number of Posts
4. Linking HashTags
5. Text / Font Size customization using CSS
6. Support for Custom CSS


<h4>Shortcode :</h4>
1. By Page Username <code>[classic-facebook-feed username="anshullabs" post_limit="10"]</code> 
2. By Page Id <code>[classic-facebook-feed user_id="566809570004880" post_limit="15"]</code>

 Here <code>post_limit</code> is display post limit. default value is 10 post.

== Installation ==
1. Upload classic-facebook-feed.zip to the /wp-content/plugins/ directory
2. Unzip classic-facebook-feed.zip
3. Activate the plugin through the 'Plugins' menu in WordPress
4. You will find 'Classic Facebook Feed' under setting menu in your WordPress admin panel.
